⚙️ Firebase 연동 설정 방법 (최초 1회)
  1. Firebase 콘솔에서 프로젝트 생성 (구글 계정으로 로그인).
  2. 좌측 빌드 → Authentication → 시작하기 → Google 사용 설정.
  3. 빌드 → Firestore Database → 데이터베이스 만들기 → 프로덕션 모드 선택 (테스트 모드는 30일간 데이터 전체 공개됨).
  4. 프로젝트 설정(⚙️) → 내 앱 → 웹(</>) 추가 → firebaseConfig 값을 복사.
  5. 아래 칸에 붙여넣고 저장 버튼을 누르세요 (자동으로 새로고침됩니다).
  6. Authentication → Settings(설정) 또는 Sign-in method(로그인 방법) 탭의 승인된 도메인에 이 사이트 주소 추가.

💾 데이터
