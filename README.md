# 컨시어지 메이트 (Concierge Mate)

호텔 컨시어지·프런트 직원을 위한 **설치 없이 브라우저에서 바로 쓰는** 업무 보조 웹앱입니다.
단일 HTML 파일 + 바닐라 JS로 만들어졌고, 데이터는 기기에 저장되며 구글 로그인 시 여러 기기에 자동 동기화됩니다.

> 제주 더본호텔(중문) 컨시어지 업무를 가정해 만들었지만, 어느 호텔에서나 설정만 바꿔 쓸 수 있습니다.

## 주요 기능

- 🏠 **홈 대시보드** — 대기/진행중/완료/미확인 인계 현황, 다가오는 예약 요청, 호텔 안내 요약
- 📋 **요청 보드** — 객실별 게스트 요청 접수·상태관리(대기→진행중→완료), 긴급 표시, **예약시간 알림**(10분 전·시간 경과 시 소리+알림)
- 🔄 **인수인계** — 교대 전달사항 기록·확인, **교대 브리핑 자동 생성 → 원터치 복사**(카톡 붙여넣기용)
- 🗺️ **제주 가이드** — 주변 명소(기본 10곳 내장), 즐겨찾기, 네이버/구글 지도 바로가기, 게스트에게 큰 화면으로 보여주기
- 💬 **외국어 회화 카드** — 영어·일본어·중국어 응대 문구 26종, **음성 재생(TTS)**, 탭하면 전체화면 표시, 직접 추가
- 💱 **환율 계산기** / 🚕 **택시 게스트 카드**(기사에게 보여줄 큰 주소 화면)
- ✅ **근무 체크리스트**(매일 자정 자동 초기화) / 📌 **메모**
- 🎤 **음성 입력** — 요청·인계·메모를 말로 받아쓰기(한국어)
- 📊 **엑셀 / 📑 PPT 보고서** — 현재 데이터를 보고서 형식으로 내보내기
- 🌙 **다크 모드**(야간 자동) · ☁️ **구글 로그인 + 기기 간 동기화**(Firebase) · 💾 백업/복원

## 사용 방법

1. 배포된 주소(아래 GitHub Pages)를 휴대폰/PC 브라우저에서 엽니다.
2. 휴대폰: 공유 → **홈 화면에 추가**하면 일반 앱처럼 전체화면으로 실행됩니다.
3. 설정(⚙️)에서 호텔 정보(체크인/아웃·조식·와이파이·주소 등)를 입력하세요.

## 구글 로그인 + 기기 간 동기화 설정 (선택, 최초 1회)

여러 기기(휴대폰·데스크탑)에서 같은 데이터를 쓰려면 무료 Firebase 프로젝트를 연결합니다.

1. [Firebase 콘솔](https://console.firebase.google.com)에서 프로젝트 생성 (구글 계정 로그인).
2. **빌드 → Authentication → 시작하기 → Google** 로그인 사용 설정.
3. **빌드 → Firestore Database → 데이터베이스 만들기**.
4. **프로젝트 설정(⚙️) → 내 앱 → 웹(`</>`)** 추가 후 `firebaseConfig` 값 복사.
5. 앱의 **설정 → 클라우드 로그인 → Firebase 연동 설정 방법**을 펼쳐 그 칸에 붙여넣고 저장.
6. **Authentication → Settings → 승인된 도메인**에 배포 주소 도메인(예: `사용자.github.io`)을 추가.
7. 새로고침 후 우측 상단 👤 버튼으로 구글 로그인.

> Firebase 설정값(apiKey 등)은 클라이언트 공개용 식별자라 외부에 노출돼도 안전합니다. 실제 보안은 Firestore 보안 규칙과 승인된 도메인으로 제어됩니다.

### 권장 Firestore 보안 규칙

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{uid} {
      allow read, write: if request.auth != null && request.auth.uid == uid;
    }
  }
}
```

## 데이터·개인정보

- 로그인 전: 모든 데이터는 이 기기 브라우저(localStorage)에만 저장됩니다.
- 로그인 후: 본인 구글 계정의 Firestore 문서(`users/{uid}`)에만 저장·동기화됩니다.
- 비밀번호 등 민감정보는 저장하지 않습니다.

## 기술 메모

- 단일 `index.html` (HTML/CSS/바닐라 JS), 빌드 도구 없음.
- 엑셀/PPT는 사용 시에만 CDN에서 [SheetJS](https://sheetjs.com), [PptxGenJS](https://gitbrent.github.io/PptxGenJS/)를 지연 로드(오프라인 시 엑셀은 CSV로 대체).
- 음성 입력/재생은 브라우저 Web Speech API(크롬·엣지·안드로이드 크롬 권장).
